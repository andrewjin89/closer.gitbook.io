# 잡담 노드 👩🏻‍🔬

![&#xC7A1;&#xB2F4; &#xB178;&#xB4DC;](../../../.gitbook/assets/2019-11-19-1.04.38.png)

잡담 노드는 사용자의 발화에 대해 자동생성된 일상회화를 반환하는 노드입니다.

{% hint style="info" %}
현재 잡담 노드는 핑퐁\(PINGPONG\) 서비스를 통해 제공됩니다.  
[https://pingpong.us/](https://pingpong.us/)
{% endhint %}
