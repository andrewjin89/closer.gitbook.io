# 개요

CLOSER는 쉽고 빠르게 챗봇을 만들어 고객응대를 자동화하고, 상담원이 보완해 완성도를 높일 수 있는 편리하고 효율적인 채팅상담 솔루션 입니다.

## Builder

빌더를 활용해 쉽고 빠르게 봇을 만들 수 있습니다.
