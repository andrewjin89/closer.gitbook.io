# 노드\(Node\)

챗봇을 설계하기 위한 가장 기본 구성 요소입니다.

다양한 기능별 노드들을 제공합니다.

노드들을 연결해 대화를 구성할 수 있습니다.

---

## 노드 팔레트

기능별 노드를 제공합니다.

* 사용자 입력 요청 노드

* 메시지 응답 노드

* 플로우 연결 노드

* 양식 입력 요청 노드

* HTTP 요청 노드

* 결제 노드

* 구글 스프레드시트 노드

* 파라미터 설정 노드

* 상담원 호출 노드

* 자연어 처리 노드

---

## 새로운 노드 추가하기

팔레트에서 원하는 노드를 골라 캔버스에 끌어다 놓기\(drag&drop\)
