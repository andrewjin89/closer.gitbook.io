# 상담하기

![](../../.gitbook/assets/openbeta_chat_%20%281%29.png)

* CLOSER Chat에 로그인하면 상담하기 화면으로 바로 진입합니다.
* 동일한 아이디로 중복으로 접속할 수 없으며, 중복으로 접속하는 경우, 팝업이 나타나고, 확인을 누르면 팀 선택 화면으로 이동합니다.
* 상담 화면은 사이드 바, 대화 목록, 채팅창 영역, 상담원 패널 4가지 영역으로 구분되어있습니다.
